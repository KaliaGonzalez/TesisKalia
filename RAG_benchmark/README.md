# RAG_benchmark

Undergraduate project of the students Juan Diego Ortega and Nicolás Díaz, under the supervision and direction of Professor Mario Linares. 

This repository contains the code used by the students for the execution of local LLM using different tools for its correct operation (Ollama, LangChain, ChromaDB, etc.)

The project is in experimental phase and the information and documents used in this benchmark are extracted from the FAC web page, so they are free to download.
